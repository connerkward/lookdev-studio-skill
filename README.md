# lookdev

**lookdev** is a Claude Code skill that spins up a live, **human-in-the-loop web studio** to tune AI-generated output by eye — sliders, color pickers, drag handles, inline markup — then bakes the chosen state into reproducible assets or code. Instead of guessing values from a screenshot, you *dial them in*.

> **What this is:** A new interaction mode for Claude Code (which is text-first): a real-time visual studio with a human in the loop, not a one-shot guess.

## Install (Claude Code plugin)

```
/plugin marketplace add connerkward/lookdev-studio-skill
/plugin install lookdev@lookdev
```

Or install the whole set: `/plugin marketplace add connerkward/connerkward-skills`. Or drop this repo's `SKILL.md` into your agent's skills directory.
